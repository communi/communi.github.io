Communi

IRC framework

Communi is a cross-platform IRC framework written with Qt and C++.

Repositories

The Communi project is hosted on GitHub.

libcommuni

The IRC framework.

communi-meego

An IRC client for MeeGo.

communi-desktop

An IRC client for desktop environments.

communi-znc

A Communi ZNC module.
